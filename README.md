# 使用说明

安装插件后，发送 `QQ估价` 即可估算自己QQ号的价值

如果想查询别人QQ的价值，只需要在命令后面艾特对方 或者输入对方QQ号 即可查询

> 如果需要更稳健的运行，配置好 `bot自己的qq号` 可以更精确的处理艾特情况
